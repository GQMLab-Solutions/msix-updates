Our installation script handles everything for you automatically.

Instructions:

  1. Download the installer using the button above

  4. When installation is complete, you'll find GQM Lab Manager in your Start menu

Option 2: Manual Installation

Install by downloading and installing the files individually.

Download Certificate Download Application

Instructions:

  1. Download both files using the links above
  2. Install the certificate:
    • Double-click the GQMLab_Simple.cer file
    • Click "Install Certificate"
    • Select "Local Machine" and click "Next"
    • Select "Place all certificates in the following store"
    • Click "Browse" and select "Trusted Root Certification Authorities"
    • Click "OK" → "Next" → "Finish"
  3. Install the application:
    • Double-click the gqm_lab_manager.msix file
    • The application will install automatically

Finding and Opening GQM Lab Manager

  • 1

    Using Windows Start Menu

    1. Click the Windows Start button in the bottom left corner
    2. Type "GQM Lab Manager" in the search box
    3. Click on the GQM Lab Manager app icon
  • 2

    Creating a Desktop Shortcut (Optional)

    1. Find GQM Lab Manager in the Start menu
    2. Right-click on the app and select "More" → "Open file location"
    3. Right-click the shortcut in the opened folder
    4. Select "Copy"
    5. Right-click on your desktop and select "Paste shortcut"

System Requirements

  • Operating System: Windows 10 or Windows 11
  • Processor: 64-bit processor
  • Memory: 4 GB RAM minimum
  • Storage: 100 MB available storage
  • Permissions: Administrator rights during installation

Need Help?

If you're experiencing installation issues:

Common Issue: If you see an error about an untrusted certificate, make sure you've installed the certificate to the "Trusted Root Certification Authorities" store for "Local Machine" (not "Current User").

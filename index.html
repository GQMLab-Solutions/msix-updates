Installation Instructions:

  1. Download the ZIP file using the button above
  2. Extract all files to a folder on your computer
    Right-click the ZIP file and select "Extract All..."
  3. Install the certificate:
    • Double-click on the GQMLab_Simple.cer file
    • Click "Install Certificate"
    • Select "Local Machine" and click "Next"
    • Select "Place all certificates in the following store"
    • Click "Browse" and select "Trusted Root Certification Authorities"
    • Click "OK" → "Next" → "Finish"
  4. Install the application:
    • Double-click the gqm_lab_manager.msix file
    • The application will install automatically
  5. Start using GQM Lab Manager - See the "Launching the Application" section below

Option 3: Manual Installation

If you prefer to download and install files individually:

Step 1: Download these files

Download Certificate Download MSIX Package

Step 2: Install the certificate

  1. Double-click the downloaded certificate file
  2. Click "Install Certificate"
  3. Select "Local Machine" and click "Next"
  4. Select "Place all certificates in the following store"
  5. Click "Browse" and select "Trusted Root Certification Authorities"
  6. Click "OK" → "Next" → "Finish"

Step 3: Install the application

  1. Double-click the downloaded MSIX file
  2. The application should install automatically

System Requirements

  • Operating System: Windows 10 (version 1809 or later) or Windows 11
  • Processor: 64-bit processor
  • Memory: 4 GB RAM minimum
  • Storage: 100 MB available storage
  • Permissions: Administrator rights during installation

Launching the Application

After successful installation, you can access GQM Lab Manager using any of these methods:

From the Start Menu

  1. Click the Windows Start button ⊞ in the bottom left corner
  2. Scroll through the app list or type "GQM Lab Manager" in the search box
  3. Click on the GQM Lab Manager app icon
GQM Lab Manager in Windows Start Menu

Using Windows Search

  1. Press the Windows key ⊞ on your keyboard
  2. Type "GQM Lab Manager"
  3. Click on the app when it appears in search results

From Desktop Shortcut

If you used the Installation Wizard (Option 2), a desktop shortcut was automatically created for you. Simply double-click this shortcut to launch GQM Lab Manager.

If you'd like to create a desktop shortcut manually:

  1. Find GQM Lab Manager in the Start menu
  2. Right-click on the app and select "More" → "Open file location"
  3. Right-click the shortcut in the opened folder
  4. Select "Copy"
  5. Right-click on your desktop and select "Paste shortcut"

Need Help?

If you're experiencing installation issues or have questions about GQM Lab Manager:

Common Installation Issues

Certificate Trust Error

If you see a message about untrusted certificates, make sure you've installed the certificate to the "Trusted Root Certification Authorities" store for "Local Machine" (not "Current User").

Cannot Find Application After Installation

If you can't find GQM Lab Manager after installation, try searching in the Start menu or check if Windows is still finishing the installation in the background.

Security Software Blocking Installation

Some security software may block the installation. Try temporarily disabling your antivirus or adding an exception for GQM Lab Manager.

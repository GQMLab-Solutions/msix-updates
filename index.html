Important Note:

Some users may encounter issues with the automatic installer due to system security settings or path configurations. If the installer doesn't work for you, please use the Manual Installation (Option 2) below, which works reliably on all systems.

Option 2: All-in-One Package (Recommended)

Download our complete package that contains all required files:

Download Complete Package

Instructions:

  1. Download the ZIP file
  2. Extract all files to a folder
  3. Right-click on the GQMLab_Simple.cer file and select "Install Certificate"
  4. Select "Local Machine" and click "Next"
  5. Select "Place all certificates in the following store"
  6. Click "Browse" and select "Trusted Root Certification Authorities"
  7. Click "OK" → "Next" → "Finish"
  8. Double-click the gqm_lab_manager.msix file to install the application

Option 3: Individual Files

If you prefer to download files individually:

1 Download the certificate and the MSIX package:

Download Certificate Download MSIX Package

2 Install the certificate:

  • Double-click the downloaded certificate file
  • Click "Install Certificate"
  • Select "Local Machine" and click "Next"
  • Select "Place all certificates in the following store"
  • Click "Browse" and select "Trusted Root Certification Authorities"
  • Click "OK" → "Next" → "Finish"

3 Install the application:

  • Double-click the downloaded MSIX file
  • Follow the installation prompts

System Requirements

  • Windows 11
  • 64-bit processor
  • 4 GB RAM minimum
  • 100 MB available storage

Launching the Application

After successful installation, you can access GQM Lab Manager using any of these methods:

From the Start Menu

  1. Click the Windows Start button in the bottom left corner
  2. Scroll through the app list or type "GQM Lab Manager" in the search box
  3. Click on the GQM Lab Manager app icon

Using Windows Search

  1. Press the Windows key on your keyboard
  2. Type "GQM Lab Manager"
  3. Click on the app when it appears in search results

Create a Desktop Shortcut (Optional)

  1. Find GQM Lab Manager in the Start menu
  2. Right-click on the app and select "More" → "Open file location"
  3. Right-click the shortcut in the opened folder
  4. Select "Copy"
  5. Right-click on your desktop and select "Paste shortcut"
